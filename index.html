Languages

Be aware the queries in this page are quite large and can take some time to appear!

You may have to reload the page to get them to load correctly.

Big Data

This wikidata query fetches:

  • The country of the language
  • Its linguistic typology
  • Its writing system
  • The number of people who speak, write, or sign in that language

All elements in this query are required to be non-null.

High Dimensional Data

This is a very similar query to the above, with the added limit of 500 languages and the following optional elements shown:

  • What the language uses capitalization for
  • Which grammatical persons the language has
  • Which grammatical genders the language has
  • Which grammatical moods the language has
  • Which grammatical tenses the language has
  • Which grammatical cases the language has
